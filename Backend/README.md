# Aplicação de Cálculos para Comércio

## Descrição
Aplicação web desenvolvida em Java (Spring Boot) para empresas do setor de comércio. Possui três telas principais:  
- **Entrada de Produtos**: cadastro de produtos com data, fornecedor, preço e prazo de pagamento.  
- **Venda de Produtos**: registro de vendas com data, cliente, produto, preço e prazo de pagamento.  
- **Gestão**: cálculo automático de PMRE, PMRV, PMPF, Ciclo Operacional, Ciclo de Caixa e Saldo Mínimo de estoque.  

O objetivo é automatizar a gestão de estoques, pagamentos e fluxo de caixa com base em dados reais.

## Tecnologias Utilizadas
- **Backend**: Java 17 + Spring Boot  
- **Frontend**: HTML, CSS, JavaScript (fetch API)  
- **Banco de Dados**: MySQL  
- **Controle de Versão**: Git + GitHub  

## Funcionalidades
- Cadastro de produtos e vendas  
- Cálculo automático de indicadores financeiros e de estoque  
- Visualização de resultados de gestão em tempo real  
- Persistência de dados no banco MySQL
